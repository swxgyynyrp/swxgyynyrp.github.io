New York State RP — Discord Guidelines

1. No Targeted Harassment; we don't allow racism, hate speech, or any unwanted conduct. Friendly banter is allowed, harassment is not.
2. Keep it Safe For Work (SFW); no pornography, nudity, or malicious links.
3. No Unsolicited Advertisement (Poaching). This includes User DMs.
4. Listen to Staff. Receiving 1-2 warns in chats will result in a temp-mute.
5. Swearing is generally permitted, unless it reaches the point where it is entirely aggravated.
